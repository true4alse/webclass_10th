flex-item들에게 지정하는 속성들

flex-basis

width처럼 아이템의 가로사이즈를 지정할 수 있다. 기본값은 auto이다.

플렉스아이템들이 점유하는 기본크기를 설정

flex-shrink

아이템의 크기가 줄어드는 비율

플렉스 컨테이너의 너비가 줄어들어서 아이템들이 설정된 너비보다 작아져야할 때, 줄어드는 비율을말한다.(기본값:1)-아이템들이 줄바꿈이 허용되지 않는 상황에서 유효하다.

기본값은 1이며 컨테이너 공간이 줄어들면 아이템들의 너비가 같이 같은 비율로 줄어든다.

flex-grow

플렉스 컨테이너의 공간이 여유가 있어서 남아돌 때, 남은 공간을 아이템들이 나눠갖는 비율

기본값은 0이며 남은 공간이 있어도 아이템의 크기는 커지려고 하지 않는다.

order

아이템의 보여지는 순서를 바꿀 때 사용 기본값 0

order 값이 클수록 나중에 배치된다.
