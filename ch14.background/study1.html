background

background-color

배경색상 입력

태그의 공간(테두리 안쪽 모든 공간)에 배경색을 나타낼 때 사용하는 css속성

background-image

배경이미지 입력 - url()도구 사용

배경이미지는 입력하면 자동으로 가로세로로 반복된다.

background-repeat

배경이미지가 반복되는 속성

repeat(기본값-가로 세로 반복)

no-repeat 반복안함

repeat-x 가로 반복

repeat-y 세로 반복

background-position

배경이미지의 위치를 변경할 때 사용하는 속성

background-position:x좌표 y좌표

키워드방식,%방식,픽셀방식

background-size

배경이미지의 크기를 변경

background-size:가로크기 세로크기

cover방식은 배경이미지크기를 이미지가 잘려도 태그의 공간을 가득 메우는 방식

contain방식은 배경이미지가 잘려나가지 않는 선에서 배경이미지의 크기를 최대한 키워서 넣는 방식

축약형 표현

background:배경색 이미지 반복속성 좌표 / 사이즈 순서대로 씀(순서중요!외울 것)
