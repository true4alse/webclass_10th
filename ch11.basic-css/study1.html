html태그를 꾸며주는 언어

스타일 적용 방법

태그를 꾸며주는 방법으로는 3가지가 있다.

  • 인라인스타일(inline-style) 방식 - 시작태그에 style속성을 사용하는 방식 (단점:일괄 스타일 적용이 불가능하다.)
  • 내부 스타일시트 방식 - style태그를 사용하여 스타일을 입력하는 방식(단점:모든 페이지에 일괄 스타일 적용이 어렵다.)
  • 외부 스타일시트 방식 - css파일을 연결하는 방식

내부스타일시트의 단점

html파일 안에 스타일코드가 방대하게 들어가게 되면 수정보완하는데 있어서 불편하다.

스타일코드가 html파일 안에있기 때문에 동시에 여러 html페이지의 스타일 수정이 불가능하다.

인라인스타일방식의 단점

내부스타일시트방식의 단점 처럼 여러페이지 동시 수정이 불가능하다.

시작태그에 style 이라는 속성을 사용하여 스타일을 입력하는 방식이기 때문에 태그의 가독성이 좋지 않다.
