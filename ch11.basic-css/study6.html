태그 형태의 종류

태그는 크게 형태에 따라 3가지 종류로 나뉜다.

  • 블록 레벨 태그
  • 인라인 레벨 태그
  • 인라인-블록 레벨 태그

블록레벨태그 - 특징

대상 - 강조태그와 링크태그 등과같이 사용했을 때 줄바꿈이 되지 않는 태그를 제외한 거의 모든 태그들 - 95%이상

  • 태그가 포함하는 글자 개수에 관계없이 한 폭의 너비를 다 차지한다. -> 자동으로 줄바꿈처리가 된다.
  • 가로세로 너비를 지정할 수 있다.
  • 이런 블록레벨태그들은 css속성 display:block; 으로 설정된 태그들이다.

인라인레벨 태그 - 특징

대상 - 강조태그와 링크태그 등과같이 사용했을 때 줄바꿈이 되지않는 태그들

  • 태그가 포함하는 글자 개수에 맞추어 너비가 설정된다. -> 자동으로 줄바꿈되지 않는다.
  • 인라인레벨태그들은 직사각형 형태가 아니기 때문에 가로 세로 사이즈를 지정해도 적용되지 않는다. 인라인레벨 태그들의 글자들은 사람이 필기하는 것 처럼 baseline을 맞추어 배열되기 때문에 가로 세로 사이즈를 지정할 수 없다.
  • 이런 인라인레벨 태그들은 display:inline;으로 설정 되어있는 태그들이다.

인라인블록 레벨태그 - 특징

대상 - 사용했을 때 인라인처럼 나란히 배치되고, 블록처럼 가로 세로 사이즈를 지정할 수 있는 태그들

  • 인라인레벨 태그처럼 태그가 포함하는 글자 개수에 맞추어 너비가 설정되고, 블록레벨태그처럼 가로와 세로사이즈를 지정할 수 있는 형태의 태그들
  • display:inline-block;으로 설정된 태그들
