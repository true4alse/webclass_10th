영역 구성 태그 - div

영역을 나눌 때 자유롭게 사용하는 태그

기본적으로는 컨텐츠의 맥락에 따라서 영역을 나눈다.

좌우로 나란히 배치되어 있는 컨텐츠는 영역으로 설정한다.

영역태그로부터 파생되어 나온 태그들

  • header - 홈페이지의 머릿글 컨텐츠를 만들 때 사용(로고,주메뉴)
  • footer - 홈페이지의 꼬리말 컨텐츠를 만들 때 사용(하단로고,회사정보,운영정책 등)
  • main - 홈페이지의 전체 메인컨텐츠를 만들 때 사용(main태그는 1개여야만 함)
  • section - 구획을 나눌 때 자유롭게 사용
  • article - 구획을 나눌 때 자유롭게 사용(본문과 완전히 차별된 다른내용)
  • nav - 홈페이지의 메뉴영역을 구성할 때 사용
  • aside - 홈페이지 안에서 독립적인 사이드 컨텐츠를 만들 때 사용
  • figure - 이미지나 영상에 대한 영역구성을 할 때 사용 - figure태그안에서 figcaption 이라는 태그를 사용하여 이미지에 대한 제목이나 설명을 구성할 수 있다.

class/id

class

class속성은 태그에게 자유롭게 설정할 수 있는 별명이다. 2개이상 여러개의 클래스명을 지어줄 수 있다.

클래스명을 2개이상 여러개를 지어줄 때에는 띄어쓰기로 구분하여 속성값을 입력한다. 현재 이 태그는 클래스이름을 4개 가지고 있다

클래스명을 사용할 때는 특수문자를 사용해선 안된다.(_,-는 사용가능)

클래스명을 사용할 때 숫자로 시작하거나 한글로해서는 안된다.

클래스명은 중복사용이 가능하다.

id

아이디 속성은 태그에게 지어주는 고유이름이다.

다른태그에게 똑같은 아이디를 사용할 수 없다.

주로 겹치면 안되는 중요한 속성 값을 불러와 사용할 때 많이 사용한다.

아이디 이름도 클래스명 처럼 자유롭게 지어줄 수 있으나 위와 같이 특수문자를 쓰지 않아야하고 숫자로 시작하거나 한글로하면 안된다.

아이디는 클래스처럼 2개이상 여러개의 아이디를 지어줄 수 없다.
