너비와 높이의 최소값 최대값 설정하기

전제 : 너비와 높이가 고정 값이 아닌 변동 값일 때 사용하는 속성

  • min-width : 최소 너비 설정
  • max-width : 최대 너비 설정
  • min-height : 최소 높이 설정
  • max-height : 최대 높이 설정

단위

  • px : 화면 입자에 대한 기본 단위(절대단위)
  • rem : html태그에 적용된 글씨 크기 단위
  • em : 부모태그에 적용된 글씨 크기 단위
  • % : 부모태그 혹은 기준태그에 대한 상대단위
  • vw(viewport-width) : 브라우저 화면 너비에 대한 상대단위
  • vh(viewport-height) : 브라우저 화면 높이에 대한 상대단위
