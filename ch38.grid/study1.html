display : grid

grid컨테이너에 적용하는 속성

display:grid

display:grid가 설정된 태그는 그리드 컨테이너가 되고 그 안의 자식 태그들은 그리드 아이템이 된다.

grid-template-columns

그리드의 열의 개수와 크기를 정의한다.

1fr의 뜻은 1fraction, 비율 단위이다.

grid-template-rows

그리드의 행의 개수와 크기를 정의한다.

그리드 행의 높이는 기본적으로 포함한 컨텐츠 만큼의 크기이지만 그 크기를 정할 수도 있다.

grid-auto-rows

그리드의 행은 열의 개수에 따라 달라지므로 행 개수가 미지수일 때 전체 각 행의 높이를 설정할 때 사용하는 속성이다.

gap

그리드 셀 사이의 간격을 설정할 때 사용하는 속성이다.

띄어쓰기를 사용하여 위아래 셀 간격을 다르게 적용할 수도 있다. (gap:10px 20px)=> 위아래 간격은 10 좌우간격은 20으로 설정

justify-content

그리드 컨테이너 안에서 그리드 아이템그룹의 좌우 정렬을 조정할 수 있다.

start/center/end/stretch 4가지 값을 사용할 수 있다.

justify-items

각 그리드 아이템들의 좌우 정렬을 조정할 수 있다.

start/center/end/stretch 4가지 값을 사용할 수 있다.

align-content

그리드 컨테이너 안에서 그리드 아이템의 세로 정렬을 조정할 수 있다.

start/center/end/stretch 4가지 값을 사용할 수 있다.

align-items

각 그리드 아이템들의 세로 정렬을 조정할 수 있다.

start/center/end/stretch 4가지 값을 사용할 수 있다.

그리드 아이템들에게 사용하는 속성

grid-colum / grid-row

각 그리드 아이템들의 그리드 라인을 어디부터 어디까지 차지할지 설정할 때 사용하는 속성

그리드 라인은 1번부터 카운트한다.

grid-column : 1/3 => 그리드 열라인을 1번부터 3번까지 차지 -> 2칸 차지

z-index

값이 클 수록 앞으로 배치된다.

그리드에 사용하는 함수

repeat()함수 -> repeat(2,200px) -> 200px 200px

minmax()함수 -> minmax(100px,300px) -> 최소값은 100픽셀이고 최대값은 300픽셀
